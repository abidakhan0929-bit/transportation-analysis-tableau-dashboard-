# Transportation Analysis Dashboard (Tableau + JSON Dataset)

This project explores different types of transportation available between various Indian cities.  
I created a custom **JSON dataset** with 20 sample trips and used **Tableau Public** to design an interactive dashboard that highlights important travel patterns and fare insights.
